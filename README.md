## 📖 Overview

**CRUD-BASIC-PHP** is a lightweight and efficient web application demonstrating how to implement AJAX-powered CRUD (Create, Read, Update, Delete) operations using **Core PHP**, **MySQL**, **JavaScript**, and **AJAX**.

The project focuses on delivering a smooth user experience by performing database operations asynchronously, eliminating unnecessary page refreshes and improving responsiveness.

Whether you're learning PHP development or building a foundation for larger applications, this project provides a practical implementation of real-world CRUD workflows.

## 🚀 Getting Started

### 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/CRUD-BASIC-PHP.git
```

### 2️⃣ Move Project

Place the project inside:

```bash
xampp/htdocs/
```

or

```bash
wamp/www/
```

---

### 3️⃣ Create Database

Open **phpMyAdmin**

Create a database:

```sql
CREATE DATABASE crud_basic_php;
```

Import the provided SQL file.

---

### 4️⃣ Configure Database

Update your database credentials:

```php
$host = "localhost";
$user = "root";
$password = "";
$database = "crud_basic_php";
```

---

### 5️⃣ Run Application

Open:

```text
http://localhost/CRUD-BASIC-PHP
```
